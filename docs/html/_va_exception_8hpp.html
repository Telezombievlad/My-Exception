Detailed Description

A little more convinient tool for throwing exceptions.

VaException is just like std::exception, but it gives more information on the error (it allows you to pass line, function and file, in which the exception was thrown). To learn more watch Exception docs.

Warning
VaException is currently (and forever) in alpha state, so bugs can occur.
