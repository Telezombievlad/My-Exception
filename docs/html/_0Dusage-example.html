VaException  17.01.09
That's just like std::exception but BETTER
@usage

The Exception Exception is just like std::exception, but it gives more information on the error (func, line, file), Exception::what() can print arguments in any order ([msg, line, func] or [func, msg, line]), Exception is inherited from std::exception (thus, it is a std::exception), Exception supports formatted strings, just like std::printf.

See also
operator""_msg(), operator""_file(), operator""_func(), operator""_line(), _wrappers::ArgMsg::ArgMsg(), VAEXC_POS

try

using namespace VaExc;
// ArgMsg is implemented via std::sprintf, so formatting is the same.
throw Exception(ArgMsg("Error code: %d", error_code), VAEXC_POS);

throw Exception("Exception occured!!!", exc);
